Enigma Machine

 

The Enigma machine is a rotating electromechanical encrypting device which was invented by a German electrical engineer, called Scherbius in 1918, after Babbage and Kasiski managed to break the Vigenere cipher by using frequency analysis techniques at the end of the 19th century. Early usage of the Enigma began in 1920, mainly for commercial purposes. Later, it was used by military services of many countries, such as Germany before and during the WWII.

Main components

Keyboard: Consists of 26 keys representing the letters of the alphabet.

Lampboard: Consists of 26 glow lamps. When each of them illuminates, it indicates a letter by lighting a small window printed with a letter of the alphabet.

Plug-board: Consists of 26 sockets, each of which had a letter printed adjacent to it. The sockets are connected using ten leads, with a plug at each end. Plugging a lead into two sockets, for example A and B sockets results in swapping them. Therefore, a current which represents A as it enters the plug-board, represents B when it emerges.

Likewise, a current which represents B as it enters the plug-board, represents A when it emerges. Furthermore, if nothing is plugged into a letter socket, the letter is connected directly to the letter entry position to the right rotor. Using ten leads, leaves six letters unplugged. As a result, the number of permutations obtained by this arrangement is greater than the number of permutations plugging all the 26 letters.

A set of rotors (Scramblers):

Early designs of the Enigma used five rotors, namely: I, II, III, IV, and V, with more rotors added later for enhanced security. Only three are selected (or four in the case of the navy Enigma) and used. These rotors can be used interchangeably and in many possible orders (5 x 4 x 3 = 60 permutations).  Around the circumference of each rotor, there are alphabetical labels (A to Z) or sometimes also numbers from 1 to 26.

Pressing a letter key causes the right rotor  to move a step, which in turn will causes the middle rotor and left rotor to move as well, but after a certain number of moves in a way similar to the vehicle odometer, but not exactly the same because the rotors turnover at different positions. In addition, some rotors have more than one turnover positions. Furthermore, the middle rotor moves two steps forward, instead of one, if it is in its own turnover position. This is called double-stepping.

For the three-rotor Enigma, the right rotor moves a step every keystroke, the middle rotor moves a step every 26 keystrokes, and the left rotor turns over after 25 x 26 = 650 keystrokes instead of 26 x 26 x 26 = 676  because of the double-stepping of the middle rotor. Hence, the total machine period is 26 x 25 x 26 = 16900. Practically, messages were about 250 letters to prevent any possibility of repeating any rotor permutation in a single transmission, which makes it very difficult for cryptanalysts to use brute force to crack the code.

The core of each rotor consists of crossed inner wiring that maps the input letter into some other letter, or in some cases a letter may be mapped to itself. Each rotor has a unique internal wiring pattern, which act as substitution encryption. The complexity of the Enigma encryption stems from changing rotor positions.

Entry (static) Rotor:

 Its main function is to act as a connector for the output from the plugboard to the right rotor, providing a circular connection to the scrambler unit.

Reflector:

This is a fixed single-sided rotor which directs the electrical current back in the opposite direction, performing the inverse permutation, as the current flows from left to right.  Simply, each letter can be connected to any other letter in pairwise. So if A is connected to B, this means that B is also connected to A. As a result, the encryption process is reciprocal. This means that the same machine can be used for both encrypting and decrypting, provided the same settings are used. Another important result is that a letter can never be encrypted to itself. There are two main types of reflectors namely: reflector B and reflector C. These two reflectors have different inner wiring patterns.
